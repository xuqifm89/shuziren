# 剧本创作与分镜拆解 Skill

这个 skill 提供从创意构思到分镜表的完整工作流，覆盖剧本创作和分镜拆解两大核心能力。

## 功能概览

| 功能 | 说明 |
| --- | --- |
| 剧本创作 | 根据题材、风格、时长要求创作完整剧本 |
| 分镜拆解 | 将剧本逐场拆解为可执行的分镜表 |
| 剧本改写 | 对已有剧本进行风格调整、时长压缩、角色替换等 |
| 分镜优化 | 对已有分镜表进行节奏调整、镜头优化 |
| 剧本→分镜一键转换 | 从剧本直接生成分镜表 |
| 分镜→剧本反推 | 从分镜表反向生成剧本文档 |

## 默认工作流

1. 确认创作需求：题材类型、目标时长、角色设定、风格偏好
2. 参考 `references/script-format.md` 选择剧本格式
3. 创作剧本（含场景描述、对白、动作指示）
4. 参考 `references/storyboard-format.md` 选择分镜格式
5. 将剧本拆解为分镜表
6. 参考 `references/workflow.md` 进行质量检查和迭代优化

## 支持动作

### 剧本创作

- 从零创作完整剧本
- 根据关键词/主题生成剧本大纲后展开
- 改写已有剧本（换风格、压缩时长、替换角色）
- 多角色对话剧本生成
- 旁白/解说词撰写

### 分镜拆解

- 将完整剧本拆解为分镜表
- 按场景拆分镜头
- 为每个镜头指定景别、角度、运动方式
- 估算每个镜头时长
- 标注转场方式
- 添加画面描述和构图建议

## 剧本类型支持

| 类型 | 说明 | 典型时长 |
| --- | --- | --- |
| 短视频剧本 | 抖音/快手等平台竖屏短视频 | 15s-3min |
| 数字人播报 | 数字人口播+图文配合 | 30s-5min |
| 广告片 | 品牌宣传/产品推广 | 15s-60s |
| 微电影 | 完整叙事短片 | 3min-15min |
| 知识科普 | 知识讲解/教程演示 | 1min-10min |
| 情景剧 | 多角色对话情景 | 1min-5min |

## 命令选择建议

- 用户要"写剧本/写脚本/写文案" → 剧本创作
- 用户要"拆分镜/做分镜表/镜头拆解" → 分镜拆解
- 用户提供了剧本要"转分镜/出分镜" → 剧本→分镜一键转换
- 用户要"改一下剧本/换个风格" → 剧本改写
- 用户提供了分镜要"写剧本/出文案" → 分镜→剧本反推

## 执行前检查

- 确认目标平台（竖屏/横屏）影响分镜构图建议
- 确认目标时长，直接影响剧本字数和分镜数量
- 确认是否有角色设定或形象限制（如数字人类型）
- 确认是否有品牌调性或风格指南需要遵循
- 如果是改编，确认原素材的版权和使用范围

## 输出规范

### 剧本输出

- 使用标准剧本格式（参见 `references/script-format.md`）
- 场景编号连续，不跳号
- 角色名统一，同一角色不出现不同称呼
- 对白与动作描述清晰分离
- 标注预计时长

### 分镜输出

- 使用标准分镜表格式（参见 `references/storyboard-format.md`）
- 每个镜头独立编号
- 景别、角度、运动方式使用标准术语
- 画面描述具体可执行，避免模糊表述
- 时长估算合理（短视频单镜头通常 2-8 秒）
- 转场方式明确标注
