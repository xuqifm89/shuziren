# 工作流参考

## 工作流一：从零创作（完整流程）

适用于从创意构思到分镜表的完整创作过程。

```
步骤1：需求确认
  ├── 确认视频类型（短视频/广告/数字人/微电影）
  ├── 确认目标平台（竖屏/横屏）
  ├── 确认目标时长
  ├── 确认风格调性
  ├── 确认角色设定
  └── 确认核心信息/主题

步骤2：创意构思
  ├── 确定核心冲突/卖点/信息
  ├── 构思开场钩子（前3秒必须抓住注意力）
  ├── 规划叙事结构（起承转合/问题-方案/对比-反转）
  └── 输出：创意大纲（3-5句话概括）

步骤3：剧本创作
  ├── 按选定的剧本格式撰写
  ├── 控制字数（参考 script-format.md 中的字数/时长对照表）
  ├── 确保对白自然口语化
  ├── 标注关键动作和情绪
  └── 输出：完整剧本

步骤4：剧本自检
  ├── 时长是否匹配目标
  ├── 开场是否有钩子
  ├── 节奏是否有起伏
  ├── 对白是否自然
  ├── 核心信息是否传达清晰
  └── 修改至满意

步骤5：分镜拆解
  ├── 将剧本逐场拆分为镜头
  ├── 为每个镜头指定景别、角度、运动方式
  ├── 估算每个镜头时长
  ├── 标注转场方式
  ├── 添加画面描述
  └── 输出：分镜表

步骤6：分镜自检
  ├── 总时长是否匹配
  ├── 镜头节奏是否合理（快慢交替）
  ├── 景别是否有变化（避免连续相同景别）
  ├── 画面描述是否具体可执行
  ├── 转场是否自然
  └── 修改至满意
```

## 工作流二：剧本→分镜快速转换

适用于已有剧本，只需拆分镜的场景。

```
步骤1：读取剧本
  └── 确认剧本格式和内容完整度

步骤2：场景切分
  ├── 按场景/地点变化切分
  ├── 按时间变化切分
  ├── 按情绪/节奏变化切分
  └── 标注每个场景的起止位置

步骤3：镜头拆分
  ├── 每个场景拆分为2-5个镜头
  ├── 开场场景优先使用远景/全景交代
  ├── 对话场景使用正反打（近景/中景交替）
  ├── 情绪高潮使用特写/推镜头
  └── 动作场景使用多角度快速剪辑

步骤4：填充分镜细节
  ├── 景别、角度、运动方式
  ├── 画面描述
  ├── 对白/旁白对应
  ├── 时长估算
  └── 转场方式

步骤5：时长校验
  ├── 各镜头时长之和是否匹配目标
  ├── 是否需要压缩或扩展
  └── 调整至总时长合理
```

## 工作流三：分镜→剧本反推

适用于已有分镜表，需要生成剧本文档的场景。

```
步骤1：读取分镜表
  └── 确认分镜信息完整度

步骤2：提取叙事线索
  ├── 从画面描述中提取场景信息
  ├── 从对白/旁白中提取文案内容
  ├── 从景别和运动中推断情绪节奏
  └── 从转场中推断场景关系

步骤3：重组剧本
  ├── 按场景归并镜头
  ├── 将零散对白组织为连贯对话
  ├── 补充场景描述和动作指示
  └── 按选定剧本格式输出

步骤4：润色
  ├── 确保对白连贯自然
  ├── 补充分镜中省略的过渡
  └── 统一角色称呼和风格
```

## 工作流四：剧本改写/优化

适用于对已有剧本进行调整。

```
步骤1：分析原剧本
  ├── 提取核心叙事结构
  ├── 识别可优化的部分
  └── 确认改写目标（换风格/压缩时长/替换角色/调整节奏）

步骤2：执行改写
  ├── 风格调整：保持叙事结构，改写对白和描述风格
  ├── 时长压缩：删减次要场景，精简对白，加快节奏
  ├── 时长扩展：增加细节场景，丰富对白，放慢节奏
  ├── 角色替换：保持叙事逻辑，替换角色设定和对白
  └── 节奏调整：重新安排冲突和高潮的位置

步骤3：对比校验
  ├── 核心信息是否保留
  ├── 改写目标是否达成
  └── 整体质量是否提升
```

## 质量检查清单

### 剧本质量

- [ ] 开场3秒内有钩子/吸引力点
- [ ] 核心信息/主题明确
- [ ] 对白口语化、自然
- [ ] 节奏有起伏（不是平铺直叙）
- [ ] 字数与目标时长匹配
- [ ] 角色行为逻辑自洽
- [ ] 结尾有记忆点或行动引导

### 分镜质量

- [ ] 景别有变化（避免连续3个以上相同景别）
- [ ] 镜头运动有目的（不是为动而动）
- [ ] 画面描述具体可执行（不是"好看的画面"）
- [ ] 时长估算合理（短视频单镜头2-8秒）
- [ ] 总时长与目标匹配
- [ ] 转场方式与内容节奏匹配
- [ ] 音效/音乐标注合理

### 剧本与分镜一致性

- [ ] 分镜覆盖剧本所有内容
- [ ] 对白/旁白与剧本一致
- [ ] 场景描述与分镜画面对应
- [ ] 情绪节奏在分镜中得到体现
