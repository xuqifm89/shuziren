# 剧本格式参考

## 格式一：标准影视剧本格式

适用于微电影、情景剧等叙事类内容。

```
剧本标题：《XXX》
类型：[短视频/广告片/微电影/情景剧]
预计时长：[XX分钟]
角色列表：
  - 角色A：[简要描述]
  - 角色B：[简要描述]

---

场景1  [内景/外景] [地点] [时间]

[场景描述/动作指示]

角色A
（动作/情绪提示）
对白内容

角色B
（动作/情绪提示）
对白内容

---

场景2  [内景/外景] [地点] [时间]
...
```

### 示例

```
剧本标题：《迟到的早餐》
类型：短视频
预计时长：1分30秒
角色列表：
  - 小明：上班族，总是迟到
  - 老板：严厉但心软

---

场景1  内景  公司会议室  上午

会议室里，同事们已经坐好。门口传来急促的脚步声。

小明
（气喘吁吁，推门而入）
对不起对不起！地铁又故障了！

老板
（看了一眼手表，面无表情）
第三次了，小明。

小明
（低头，尴尬地挠头）
我...我保证下次一定...

老板
（从桌下拿出一个早餐袋，推过去）
先吃吧，别饿着。下次再迟到，早餐就没了。

小明
（惊讶，接过早餐袋）
谢谢老板...

---

场景2  ...
```

## 格式二：短视频脚本格式

适用于抖音/快手等竖屏短视频，节奏紧凑。

```
脚本标题：《XXX》
平台：[抖音/快手/小红书/通用]
时长：[XX秒]
风格：[搞笑/情感/科普/种草/剧情]
BGM建议：[音乐风格/具体曲目]

---

[00:00-00:03] 开场钩子
画面：[画面描述]
旁白/对白：[文案内容]
字幕：[需要展示的字幕]

[00:03-00:08] 铺垫
画面：[画面描述]
旁白/对白：[文案内容]

[00:08-00:15] 冲突/转折
画面：[画面描述]
旁白/对白：[文案内容]

[00:15-00:20] 高潮
画面：[画面描述]
旁白/对白：[文案内容]

[00:20-00:25] 结尾/反转
画面：[画面描述]
旁白/对白：[文案内容]

[00:25-00:30] 引导互动
画面：[画面描述]
字幕：[引导语]
```

### 示例

```
脚本标题：《打工人的早晨》
平台：抖音
时长：30秒
风格：搞笑
BGM建议：轻快节奏感强的音乐

---

[00:00-00:03] 开场钩子
画面：闹钟显示7:30，一只手猛地拍掉闹钟
旁白：每个打工人都有同一个梦想——
字幕：你中枪了吗？

[00:03-00:08] 铺垫
画面：快速剪辑——刷牙、穿衣服、抓面包
旁白：再睡五分钟，就五分钟...

[00:08-00:15] 冲突
画面：看手机显示8:15，表情从迷糊变惊恐
旁白：结果五分钟变成了四十五分钟！
字幕：完了完了完了

[00:15-00:20] 高潮
画面：狂奔进电梯，门刚好关上
旁白：世界上最远的距离，就是我和那扇电梯门！
字幕：不——！

[00:20-00:25] 反转
画面：到工位，发现老板还没来，松一口气
旁白：结果...老板比我更会赖床。
字幕：安全上垒 ✌️

[00:25-00:30] 引导互动
画面：对着镜头比耶
字幕：你今天迟到了吗？评论区报到！
```

## 格式三：数字人播报脚本格式

适用于数字人口播类视频。

```
播报标题：《XXX》
数字人：[角色名/形象描述]
时长：[XX秒]
风格：[正式/亲切/专业/活泼]

---

[开场白]
[数字人播报内容，标注语气和表情变化]

[正文段落1]
[播报内容]
（插入画面：[配合展示的图片/视频素材描述]）

[正文段落2]
[播报内容]
（插入画面：[配合展示的图片/视频素材描述]）

[结尾]
[播报内容]
（引导语：[点赞/关注/分享等]）
```

### 示例

```
播报标题：《今日科技速递》
数字人：小智/职业女性形象
时长：60秒
风格：专业亲切

---

[开场白]
大家好，我是小智，欢迎来到今日科技速递。
（微笑，自然手势）

[正文段落1]
今天最值得关注的消息是，国产大模型再获突破！
（语气上扬，略带兴奋）
（插入画面：大模型界面截图/科技感背景）

[正文段落2]
据悉，新模型在多项基准测试中表现优异，特别是在中文理解和代码生成方面。
（语速适中，专业感）
（插入画面：测试数据对比图）

[结尾]
以上就是今天的科技速递，关注我，每天带你了解科技前沿。
（微笑，微微点头）
（引导语：点赞关注，每天见！）
```

## 格式四：广告片脚本格式

适用于品牌宣传和产品推广。

```
广告标题：《XXX》
品牌：[品牌名]
产品：[产品名]
时长：[XX秒]
调性：[高端/温暖/活力/幽默]
目标受众：[受众描述]
核心信息：[一句话核心卖点]

---

[镜头1]  [时长]
画面：[画面描述]
旁白/字幕：[文案]
音效/音乐：[声音设计]

[镜头2]  [时长]
画面：[画面描述]
旁白/字幕：[文案]
音效/音乐：[声音设计]

...

[结尾镜头]  [时长]
画面：品牌Logo + Slogan
字幕：[Slogan]
音效/音乐：[品牌音效]
```

## 字数与时长参考

| 类型 | 语速参考 | 字数/分钟 |
| --- | --- | --- |
| 新闻播报 | 较快 | 250-300字 |
| 日常口播 | 中等 | 200-250字 |
| 情感叙事 | 较慢 | 150-200字 |
| 搞笑短视频 | 快 | 250-350字 |
| 广告旁白 | 中等偏慢 | 180-220字 |
| 知识科普 | 中等 | 200-260字 |
